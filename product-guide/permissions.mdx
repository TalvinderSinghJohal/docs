## Overview

AltrumAI provides robust user management tools to help Admins control access and responsibilities across the organization.

## Accessing User Management

1. Navigate to the **User Management** tab from the Navigation Panel
2. You will see a list of all current users and their roles

<img height="200" src="images/Administration.jpg" />

## Adding a New User (Admin Only)

1. Click on the **User Management** tab
2. Click the **Invite** button
3. Enter the user's email address
4. Select the appropriate **Role** (Admin or User)
5. Click **Invite** to send the invitation

![Placeholder for invite new user screenshot]

## Editing a User Role

1. From the User Management screen, locate the user
2. Click on their **current role** to open a dropdown
3. Select the new role (Admin/User)
4. Click **Save** to apply changes

![Placeholder for editing user role screenshot]

## Editing Account Details

1. Click on the **Account Detail** tab
2. Click the **Edit** button
3. Update the necessary user information
4. Click **Save** to confirm changes

![Placeholder for editing account details screenshot]

## Changing User Status

1. Locate the user whose status you want to change
2. Use the **Status** dropdown to set the new status (e.g., Active, Inactive)
3. Click **Save** to apply the new status

![Placeholder for changing user status screenshot]

## User Roles Explained

- **Admin**: Full access to platform settings and user management
- **User**: Limited access, typically assigned to project-specific tasks

## Best Practices

- Regularly review user roles and statuses for compliance
- Only assign Admin roles to users who require elevated privileges
